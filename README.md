# SpectrumPlay

Simple audio spectrum visualizer written in C using raylib and GLSL shaders.

## Purpose

This project was created for learning purposes:

- Fast Fourier Transform (FFT)
- C programming
- real-time audio processing
- basic graphics and shaders (GLSL)
- WebAssembly via Emscripten

## Features

- waveform visualization
- frequency spectrum (FFT)
- spectrogram rendering
- adjustable gain and number of bars
- basic color modes
- desktop and web build

## Tech Stack

- C
- raylib
- GLSL shaders
- Emscripten (WebAssembly)
